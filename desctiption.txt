Задание: 

Написать программу, собирающую информацию об активности файловых вирусов. 

Часть 1. Создать интегрированный файл в формате ‘cvs’, в котором исходную информацию дополнить идентификатором компьютера. 

На вход программа получает путь к каталогу, где располагаются файлы с результатами антивирусного сканирования (например, D:\Test\Source\) и путь к каталогу для сохранения результатов (D:\Test\Result\). 

Каждая строка содержит 5 блоков информации, разделяемых точкой с запятой(их порядок варьируется в зависимости от варианта): 

UserID; ID;FileName;FileSize;IsVirus

 где: 

UserID – уникальный идентификатор компьютера, с которого получены данные. Уникальный идентификатор представляет собой первые 5 символов имени файла с результатами сканирования, полученными с конкретного компьютера. Например, имя файла, с результатами сканирования 12345678.cvs, тогда UserID для данного пользователя будет «12345». 

Остальные поля интегрированного файла с результатами идентичны полям файлов с результатами сканирования для каждого пользователя. 

Файл результатов сохраняется в каталог для сохранения результатов под именем “ScanReport.csv”. 

Часть 2. Из всех файлов в каталоге, содержащих результаты антивирусной проверки сформировать файл-отчет, содержащий данные о популярности (Filecount) и распространенности (UserCount) каждого файла среди пользователей. Каждая строчка файла отчета выглядит следующим образом (их порядок варьируется в зависимости от варианта): 

ID;UserCount;FileCount;UserVirusCount;VirusCount 

где: 

ID – уникальный идентификатор файла. В файле отчета не должно быть повторяющихся идентификаторов. Все записи уникальны. 

UserCount – количество файлов (пользователей) с результатами сканирования в которых встречается данных ID. Если ID встречается в трех файлах с результатами сканирования, то UserCount = 3. При этом, не имеет значения сколько раз конкретный ID встречается в каждом из файлов с результатами сканирования: если в одном файле ID встретился 8 раз, в другом 3, а в третьем 1, то UserCount все равно будет равен 3.

FileCount – счетчик встречаемости данного уникального идентификатора во всех файлах с результатами сканирования. Если в одном файле ID встретился 8 раз, в другом 3, а в третьем 1, то FileCount = 8 + 3 + 1 = 12. 

UserVirusCount – количество файлов (пользователей) с результатами сканирования в которых встречается данных ID, где хотя бы один раз был диагностирован вирус. 

VirusCount – счетчик встречаемости данного уникального идентификатора во всех файлах с результатами сканирования, где был диагностирован вирус. 

Полученный отчет сохранить в каталог для сохранения результатов под именем “TotalReport.csv”.
